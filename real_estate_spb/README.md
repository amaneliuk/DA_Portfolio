# Исследование недвижимости Санкт-Петербурга

## Данные
Таблица содержит 23699 записей, 22 столбца с такими данными, как цена, кол-во комнат, расположение, площадь и многими другими признаками, характеризующими квартиры.

В 14-ти столбцах содержатся пропущенные значения. Пропуски вызваны скорее всего неполным заполненим пользователями полей формы объявления.

## Задача
Необходимо провести исследовательский анализ данных и выяснить:
- Факторы влияющие на стоимость квартиры
- Цены в пригороде
- Корреляция различных факторов с ценами

## Использованные методы и библиотеки
*Предобработка данных, исследовательский анализ данных, визуализация*

**Pandas, Matplotlib**

## Результаты
Были обработаны пропуски, объекты недвижимости категоризированы, выявлены характерные зависимости цен от различных критериев и характеристик квартир. Выявлены наиболее популярные локации.
